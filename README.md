# RDE-Seg

Official repository for **RDE-Seg: Role-Disentangled Experts with Residual Routing and Anatomy Constraints for DSA Guidewire Segmentation**.

## Overview

RDE-Seg is a framework for anatomy-constrained DSA guidewire segmentation. It models the structural dependency between guidewires and vessels by combining parameter-efficient domain adaptation, role-disentangled expert decoding, residual routing, and anatomy-constrained refinement.

The current implementation is based on SAM and is designed for thin-structure medical image segmentation.
